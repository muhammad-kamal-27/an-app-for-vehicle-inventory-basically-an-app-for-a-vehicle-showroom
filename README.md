# an-app-for-vehicle-inventory-basically-an-app-for-a-vehicle-showroom
Vehicle Inventory Manager
A sleek, mobile-first inventory management application built with React Native and Expo. This app allows users to track vehicle stock, categorize listings, and monitor sales performance with real-time analytics.

🚀 Features
Inventory Tracking: Add new vehicles with titles, prices, and custom notes.

Media Integration: Support for uploading vehicle images directly from the device gallery.

Smart Categorization: Filter and sort vehicles by type (Bicycle, Motorbike, EV, Scooter, etc.).

Sales Logic: Toggle "Available" and "Sold" statuses to track business flow.

Dashboard Analytics: Real-time calculation of total stock value, revenue, and average price per unit.

Dark Mode UI: A modern, high-contrast interface designed for clarity and ease of use.

🛠️ Tech Stack
Framework: React Native

Tooling: Expo

State Management: React Hooks (useState, useEffect)

Icons & UI: Vector Icons and Custom Styled Components
